# SearchWord

It finds words within a matrix of strings from:

- right to left
- left to right
- top to bottom
- bottom to top
- top-left to bottom-right
- top-right to bottom-left
- bottom-left to top-right
- bottom-right to top-left

Returns start and end position of each searched word.

result = {
word1: {
start: [x, y]
end: [x, y]
}
word2: {
start: [x, y]
end: [x, y]
}
}
